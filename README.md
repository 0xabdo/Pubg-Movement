A simple C++ tool that updates all `Type="CrossKey" speed="0"` entries to `speed="100"` in the TVM_100.xml file inside the AndroidTbox folder.

## How to Use
1) Download and run the EXE.<br>
2) Follow the on-screen instructions — the program will automatically update the XML.<br>
3) Press Enter to close the window after it’s done.
